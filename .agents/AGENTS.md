# FinKeep Workspace Rules

## Planning Workflow
When starting a task, we plan first and document/update the plan in the [planning.md](file:///Volumes/DEV/Projects/FinKeep/FinKeep/planning.md) file at the root of the repository. After the task is completed, we update [planning.md](file:///Volumes/DEV/Projects/FinKeep/FinKeep/planning.md) to check off the progress.
